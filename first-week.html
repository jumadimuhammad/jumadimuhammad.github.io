First Week at Bootcamp

First Week at Bootcamp

26 january 2020

Hari pertama membrikan kesan pertama yang tidak semestinya, berangkat dari keyakinan dengan sejuta harapan. Dimulai dengan hari ini, tepatnya senin 20 januari 2020, hari pertama dimulainya kelas di hacktiv8. Dimulai dari perkenalan hinngga penjabaran tentang pembelajaran selama mengikuti bootcamp. Dilanjutkan dengan pembelajaran tentang html, kita mengerjakan pembelajaran pada codecademy. Html merupakan bahasa pemrograman markah yang digunakan untuk membangun sebuah halaman website yang terintegrasi dan dapat diakses melalui internet.

Hasil belajar HTML

Sekarang ini website menjadi salah satu alat yang digunakan oleh banyak orang untuk berbagai keperluan salah satunya adalah jual beli, atau yang lebih dikenal e-commerce. Pada minggu pertama ini, selain diajarkan tentang html kita juga belajar mengenai CSS. CSS merupakan sebuah alat untuk menghias html agar lebih enak untuk dilihat dan dinikmati, tujuannya tidak lain untuk menarik pengunjung website. Selain belajr tentang html dan css, diminggu pertama ini kita juga belajar tentang git dan github tujuan penggunaan git dan github ini untuk maintenance dan recovery file yang telah kita buat jika terjadi hal-hal yang tidak diinginkan. Inti dari pembelajaran diminggu pertama ini adalah pengembangan website statis yang nantinya akan dikembangkan pada pertemuan-pertemuan berikutnya.

Hasil belajar CSS

Untuk saya yang baru memulai lagi berada didepan komputer dengan waktu yang lama ini terkesan berat dan sedikit memberikan kejutan kepada tubuh dan pikiran. Tak luput dari rasa putus asa namun tekad yang lebih kuat membuat saya terus berjuang. Bukan minggu pertama yang mudah memang tapi juga bukan yang tersulit. Dalam pebangunan website yang terkesan sederhana ini pun saya membutuhkan waktu dua hari untuk bisa diakses oleh semua. Ini adalah postingan pertama saya dalam website ini sekaligus website pertama yang saya bangun di pembelajaran ini. Harapan saya tetap bisa terus dikembangkan dan menjadi baik dan bisa dilihat dan dinikmati oleh semua secara berkelanjutan.
